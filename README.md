# 🛒 Flipkart Clone

A responsive Flipkart Clone created using **HTML, CSS, and JavaScript**. This project recreates the user interface of Flipkart's homepage and showcases modern front-end development skills.

## 🚀 Features

- Responsive Design
- Modern Navigation Bar
- Search Bar
- Hero Banner
- Product Categories
- Product Cards
- Offers Section
- Footer
- Hover Effects & Animations
- Clean and User-Friendly UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
Flipkart-Clone/
│── index.html
│── style.css
│── script.js
│── images/
└── README.md
```

## 🎯 Purpose

This project was built to improve my front-end development skills by recreating the interface of one of India's largest e-commerce websites.
